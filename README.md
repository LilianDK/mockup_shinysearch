# Front-End

**Step 1: Run FastAPI**
```python
uvicorn main:app --host 0.0.0.0 --port 8001 --reload
```
The app will run at localhost:8001.

**Step 2: Run UI** 
```python
shiny run shinysearch/app.py
```
The app will run at localhost:8000.
